Stockroom Inventory System
A modular Python CLI app for managing products and stockroom locations.
• Add, edit, delete, and search products
• Create and audit stockroom locations
• Back‑stock and remove inventory
• CSV‑based persistence
• Colorized terminal output
• Fully modular architecture (Product, Inventory, Stockroom, Location, Messages, Colorize)
Run with:
python main.py
